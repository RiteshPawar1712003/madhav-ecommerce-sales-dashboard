## Madhav Ecommerce Sales Dashboard (Power BI)

This project presents an interactive Power BI dashboard developed to analyze ecommerce sales performance and generate actionable business insights. The dashboard provides a comprehensive view of key performance indicators including total sales amount, total quantity sold, overall profit, and average order value.

The dashboard enables users to explore:
- State-wise sales performance
- Category and sub-category level quantity analysis
- Monthly profit trends
- Customer-wise purchase contribution
- Payment mode distribution

Interactive filters such as quarter and state selection allow dynamic analysis, helping businesses identify sales patterns, profitable segments, and customer purchasing behavior for better decision-making.

**Tools Used:** Power BI, Excel/CSV, DAX
